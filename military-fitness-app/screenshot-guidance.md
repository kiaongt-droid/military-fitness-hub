# Screenshot Guidance

## Recommended screenshot set
- Dashboard and profile screen
- Physical Fitness Test entry screen
- Combat Fitness Test entry screen
- Performance Graphs screen
- Training plan and meal plan output screens

## Suggested presentation
- Use mobile device mockups if possible.
- Highlight the easy logging flow and summary cards.
- Show meaningful data in charts and history entries.
- Keep screenshots simple and readable.

## Android / Google Play
- Include at least 2 landscape and 3 portrait screenshots.
- Use clear labels for each screen.
- Ensure text is legible on small phone previews.

## iOS / App Store
- Provide portrait screenshots for iPhone screens.
- Add one compact screenshot for iPad if desired.
- Use the first screenshot to show the app's core value.

## Best practices
- Use consistent styling and branding.
- Keep the UI visible and not overcrowded.
- Show the profile summary, metrics, and AI-generated plan content.
- Avoid filler data; use realistic scores and workout details.
