## App name

## Short description

## Full description
Military Fitness Hub helps soldiers and fitness enthusiasts monitor profile progress, save physical and combat test results, and visualize performance over time.

Features:
- Save 3-mile run, plank, and pull-up test results
- Log Maneuver Under Fire, 880m run, and ammo can lifts
- View performance history with charts
- Generate a weekly training plan based on weak areas
- Create meal suggestions for fat loss, performance, or maintenance
- Offline and installable as a PWA

## Key features
- Timestamped fitness history for physical and combat tests
- Simple profile screen with height, weight, age, and waist size
- AI-ready training plan recommendations
- Meal plan generator with dietary restrictions
- Offline support via service worker and PWA manifest
